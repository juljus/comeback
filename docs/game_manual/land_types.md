# Land Types

There are 18 capturable land types in the game. Each has unique mana production, defenders, and special buildings.

## Universal Buildings

> "On all lands you can build: fortifications (3 levels), fletchery, and archery guild. On all lands you can replace land defenders with four strength levels of mobs."
> — Help file, line 79

All capturable lands can build:
- Fort → Citadel → Castle (fortifications)
- Fletchery → Archery Guild (archers)
- Portal (teleportation)
