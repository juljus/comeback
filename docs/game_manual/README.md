# Game Manual

Documentation for the Comeback game mechanics, extracted from the original 2004 Excel/VBA game.

## Source Hierarchy

1. **Help file** (`docs/extraction/raw/help.csv`) — Source of truth #1 for game design intent
2. **VBA code** (`docs/extraction/vba/all_modules.txt`) — Fills in mechanical details when help is silent

Both are valid sources. If they conflict, help takes priority (designer intent over implementation quirks).
